# 腓利門書「讀經四步」重新設計（以 joy.org.tw 為依據）

## 目標

重新設計「讀經四步」遊戲中的**腓利門書**內容，使全部內容（尤其「解釋」「應用」兩關）都有 joy.org.tw（真耶穌教會喜信網路家庭）文章的依據來源，並修正既有錯誤。**猶大書內容不動。**

## 依據來源（joy.org.tw）

1. 蔡凱宙〈進入聖經的世界(下)—腓利門書〉
   `https://joy.org.tw/goodnews_books_content.php?id=176`
   — 寫作背景、本書特色、中心信息「愛就是在別人的需要上看到自己的責任」、內容分析（分段大綱）、現代意義（饒恕／和好／使人和睦）。
2. 憂谷〈阿尼西母—悔改重生的人〉（喜信福音 796 期，摘自《典型在夙昔》）
   `https://joy.org.tw/goodnews_old_detail.php?num=5552`
   — 名字意義「有益處」、悔改重生、脫去舊人穿上新人（弗 4:21–24）、保羅的心上人、浪子回頭的屬靈類比。

經文一律用和合本（公有領域）。

## 錯誤修正

- 書卷代號 `mark`：**腓 → 門**（「腓」是腓立比書的簡稱；腓利門書正確簡稱為「門」）。連帶火漆封印、選單圓標、完成大印都改為「門」。
- 應用實踐項的經文出處：簡稱 **腓 → 門**，並移除單章書卷不必要的「1:」，採 joy 原文寫法（例：`門 18–19`、`門 14`）。
- 完成頁金句出處：`腓利門書 1:15–16` → `腓利門書 15–16`。
- 靜態 HTML 中殘留的預設「腓」字（waxMark、bigseal）一併改為「門」。

## 內容設計（重寫 `BOOKS.philemon`）

### ① 觀察（六何・6 題）
沿用六何架構，解說改扣 joy 蔡凱宙文：何人（保羅／提摩太；腓利門富有熱心、亞腓亞可能是妻、亞基布可能是子；阿尼西母主角）、何處（歌羅西腓利門家中教會，西 4:9）、何時（羅馬監獄、可接待朋友、盼獲釋，徒 28:30／門 22）、何事（逃奴造成損失又逃走、在羅馬信主）、為何（求接納為弟兄，16–17）、如何（不用使徒身分命令，憑愛求、親筆代償，8–9／19）。

### ② 結構分析（採 joy 內容分析的 5 段）
- A 找主題：正解＝「愛就是在別人的需要上看到自己的責任——在基督裡饒恕、接納、和好」；回饋引林後 5:18 和好福音。
- B 分段（5 塊）：問安祝福(1–3)／感謝稱讚(4–7)／愛的代求(8–16)／保證與盼望(17–22)／問安祝福(23–25)。
  - 註：joy 原文標「17–21」，其說明含「盼望再見面」＝第 22 節，故補為 17–22 以涵蓋全書。
- C 看上下文：第 16 節「不再是奴僕，乃是弟兄」的背景——回饋補入 joy 的羅馬奴隸法背景（逃奴可被鞭打、監禁、處死），凸顯「弟兄」的份量與和好福音。

### ③ 解釋（合宜／不合宜・5 則，全部有 joy 依據）
1. 【合宜】朋友寫給朋友的信、保羅唯一純私人信件，讀作憑愛心的勸與求。
2. 【不合宜】以使徒權柄命令、沒有商量餘地（joy：刻意只稱「為主被囚的保羅」）。
3. 【不合宜】18 節「歸在我帳上」＝信耶穌就能還清金錢債（斷章取義；joy：作保、預表基督代贖）。
4. 【合宜】名意「有益處」，信主後名副其實、脫去舊人穿上新人（joy〈阿尼西母〉，弗 4:21–24）。
5. 【合宜】稱「心上的人」「高過奴僕的弟兄」，主僕被弟兄關係超越、尊重人性尊嚴。

### ④ 應用
- A 分辨（命令／應許／榜樣・5 題）：接納弟兄如接納主(命令,17)；保羅代求代償如基督(榜樣,18–19)；阿尼西母悔改重生回去面對舊主(榜樣,10–12)；破裂關係在基督裡復和成更美弟兄(應許,15–16／林後5:18)；得赦免得平安、能饒恕得喜樂(應許，joy 現代意義)。
- B 今天如何實踐（5 選項，皆附 joy 依據與「門X」出處）：主動饒恕（七十個七次，門18–19）；重新接納跌倒的弟兄（以善勝惡不計前過，門16–17）；為人代求並願承擔（門10、17–19）；不勉強人給人甘心的空間（門14）；想起自己也曾像阿尼西母離開主、回轉（浪子回頭，門15–16）。
